Save Schema to LocalStorage
===========================

Schema Generator can store the data in the browser. This way, the user can continue to work on the schema, adding more fields as needed.

.. raw:: html

    <img src="./_images/D11.png" title="Saved" alt="Saved" />


Recover Data button
===================

Clicking the ``Recover Data`` button will take us right back to :doc:`fields`

Clear Data button
=================

Clicking the ``Clear Data`` button will delete all information saved in LocalStorage.
